Translator App

This semester I created a translator application using a React frontend, Node/Express backend, and Azure's translation api. The translator takes an input in any language and returns translations to English, Armenian, Russian, and Spanish (these are the languages that I am currently learning). This translator helps me learn the equivalent translations of words between languages.
